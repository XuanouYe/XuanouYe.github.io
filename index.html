Selected Projects

Graduation Project · Zhejiang University

Hybrid Additive-Subtractive Manufacturing for Alloy Casting

Developed and optimized a hybrid equipment workflow integrating 3D printing and CNC subtractive processes for alloy casting prototype validation.

  • Built Vericut simulation models for the full machine structure.
  • Designed support-free mold printing strategies.
  • Planned multi-axis control methods for additive and subtractive machining.
  • Created Python scripts for STL processing and G-code post-processing.
  • Implemented motion control and parameter tuning with Mach3 CNC.
Internship · Leo Pump Zhejiang Company

Coupled Mold-Flow and Structural Simulation for Plastic Pump Bodies

Built a co-simulation workflow to connect manufacturing process outputs with structural performance analysis for plastic pump components.

  • Led Moldflow analyses to evaluate weld lines and fiber orientation effects.
  • Mapped residual stress and fiber orientation results into ANSYS and Helius models.
  • Performed static FEA for operational mechanical performance assessment.
  • Documented findings in technical reports and co-authored an academic paper.
Research Training · Zhejiang University

Climbing In-Situ Hybrid Repair Robot

Contributed to a crawling robot platform for on-site additive and subtractive repair of large structures.

  • Wrote STM32 embedded code for motion control tests.
  • Supported hardware selection, assembly, and platform integration.
  • Designed operation and testing procedures for the prototype platform.
Course Project · Zhejiang University

CAM Software Development for 3D Printing

Developed a Python-based CAM tool for slicing, visualization, path planning, and G-code generation.

  • Implemented STL import and visualization using VTK.
  • Built slicing algorithms and layer-based path planning workflows.
